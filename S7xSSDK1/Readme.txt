The LoRaMac code base is GitHub Lora-net/LoRaMac-node v4.4.1.


LoRaWAN stack implements all regions defined in "LoRaWAN Regional Parameters v1.0.2rB" document.
Class A and Class C endpoint implementation is fully compatible with "LoRaWAN specification 1.0.2".


Support Regions : 
REGION_AS923 : For Asia 923MHz(about the detail to reference the country rule)
REGION_AU915 : For Australia 915~928MHz
REGION_CN470 : For China 470~510MHz
REGION_CN779 : For China 779~787MHz
REGION_EU433 : For Europe 433~434MHz
REGION_EU868 : For Europe 863~870MHz
REGION_IN865 : For India 865~867MHz
REGION_KR920 : For South Korea 920~923MHz
REGION_US915 : For United States 902~928MHz
REGION_US915_HYBRID : For United States 902~928MHz HYBRID
